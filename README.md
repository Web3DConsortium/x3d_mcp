# x3d_mcp
MCP server for generating, validating, and converting valid X3D content using assets and tools developed using the official X3D International Standards.

This project supports the [AI with X3D Special Interest Group](https://www.web3d.org/working-groups/ai-x3d) of the [Web3D Consortium](https://www.web3d.org).

This work will merge and improve capabilities found in
* https://github.com/ricklentz/x3d_mcp
* https://github.com/niknarra/x3d-mcp
